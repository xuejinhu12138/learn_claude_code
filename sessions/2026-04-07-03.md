## 关键概念

### `new Promise` vs `Promise.resolve` vs `async/await`
- `new Promise((resolve, reject) => {...})` → 给**异步回调**用（setTimeout、事件等）
- `Promise.resolve(x)` → 把同步值包成 Promise
- `async function` → 自动把 return 值包成 Promise，等价于 `Promise.resolve`
- `readFileSync` 是同步的，不需要 `new Promise`，直接用 `async` + try/catch

### 模块副作用
```typescript
import './readFile';  // 没有导入具体值，但执行了模块代码（触发 register）
```

### Record vs Map
| | Record<string, T> | Map<string, T> |
|---|---|---|
| JSON 序列化 | ✅ | ❌ |
| key 类型 | 仅 string | 任意 |
| 原型污染 | 有 | 无 |
| 适用场景 | string key 简单字典 | 复杂 key 或需要 size |
