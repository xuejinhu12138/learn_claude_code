### P5-6：Token 预算（tokenEstimation）
- 新建 `src/utils/tokenEstimation.ts`
- `estimateMessageTokens(message)`: 所有 content block 的字符数 / 3.5 向上取整 + role 固定 4 token + tool_calls arguments 字符数 / 3.5
- `estimateHistoryTokens(messages)`: 对所有消息求和
- 替换旧的消息数量阈值，改为 token 阈值检测

### P5-7a：消息分组（groupMessagesByApiRound）
- 新建 `src/utils/grouping.ts`
- 规则：遇到 `role === 'assistant'` 时结束当前组、开新组，新组以这条 assistant 消息开头
- 保证 tool_calls/tool_result 成对，不会在压缩时被拆散
- 测试全绿

### P5-7b：MicroCompact（工具结果截断）
- 新建 `src/utils/microCompactMessages.ts`
- 零 LLM 调用，纯截断
- `structuredClone(messages)` 克隆后修改，不影响原始消息
- 超过 `maxChars` 的 tool 消息内容截断，追加 `[输出过长，已截断，原始长度: N字符]`
- 测试全绿

### P5-7c：LLM 摘要压缩（compactConversation）
- 新建 `src/utils/compactConversation.ts`
- 三分区结构：`[system消息] + [摘要user消息] + [占位assistant消息] + [尾部tailSize轮次消息]`
- 调用 `groupMessagesByApiRound` 对非 system 消息分组，用 `groups.slice(-tailSize)` 取尾部
- 中间部分拼成文本，发给 LLM 请求摘要
- `if (newMessages.length <= tailSize + 1)` 消息太少时直接返回
- 测试全绿

### P5-7d：AutoCompact（自动触发 + 电路断路器）
- 新建 `src/utils/autoCompact.ts`
- `createAutoCompact(threshold, tailSize)` 工厂函数，返回 `{ check(messages, sendMessage) }` 对象
- `check()` 内：`estimateHistoryTokens(messages) <= threshold` → 原样返回
- 超过阈值 → 调 `compactConversation()`，成功 → `consecutiveFailures = 0`
- 失败 → `consecutiveFailures++`，达到 `MAX_FAILURES(3)` → 抛出 `"连续压缩失败"`
- `consecutiveFailures` 是 class 私有字段，在多次 `check()` 调用间持久

### P5-7e：压缩警告 Hook（compactWarning）
- 新建 `src/utils/compactWarning.ts`
- `createCompactWarning(warningThreshold, onWarn)` 工厂函数
- `check(messages)`: `estimateHistoryTokens > warningThreshold && !warningIssued` → 调 `onWarn()`，设 `warningIssued = true`（去重，只警告一次）
- `reset()`: 清除 `warningIssued` 标志
- 测试全绿

### P5-8：系统提示词构建（buildSystemPrompt）
- 新建 `src/utils/buildSystemPrompt.ts`
- 接收 `{ cwd, datetime, tools: [{name, description}][], memory? }` 对象
- 输出：角色定义 + 当前工作目录 + 当前时间 + 工具列表（`- name: description` 格式）+ 可选记忆
- `src/index.ts` 中在每次 `runAgent` 前调用，作为第一条 system 消息插入
- 测试全绿

### P5-9：端到端验证
- `src/agent.ts` 的 `runAgent()` 主循环集成三层压缩策略：
  1. `deps.compactWarning.check(history)` — 接近上限时提示
  2. `history = await deps.autoCompact.check(history, sendMessage)` — 超过阈值时摘要压缩
  3. `history = microCompactMessages(history, MAX_TOOL_RESULT_CHARS)` — 截断过长 tool 结果
- `src/index.ts` 创建 `AgentDeps`，注入系统提示词，调用 `runAgent`
